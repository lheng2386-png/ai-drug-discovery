# 研究问题与可检验假设

## 总问题

给定蛋白质序列、三维结合口袋及可选的已知配体，如何生成对目标蛋白具有潜在结合能力，同时满足类药性、可合成性、低毒性、物理合理性、多样性和新颖性的分子，并使多维评价反馈真正改善下一轮生成？

## 方法学贡献边界

跨模态对齐不是项目终点，而是反馈信号的一部分。论文的核心贡献必须落在以下至少一项：

1. 将 PLM、3D 口袋与配体信息融合为对未见靶点有效的条件表示；
2. 将 PLI、ADMET、物理合理性和合成性构造成经过校准的多目标奖励或偏好；
3. 让这些反馈进入采样或生成器优化，而不仅是事后删除低分分子；
4. 在严格隔离的未见靶点或未见家族测试中证明收益。

## 输入与输出

### 输入

- 蛋白质序列及 ESM-2/SaProt 表示；
- 结合口袋残基、坐标和口袋图；
- 可选的已知活性配体或药效团约束；
- 生成候选及其多维评价反馈。

### 输出

- 靶点条件化三维候选分子；
- PLI/亲和力、QED、SA、ADMET/毒性、物理合理性评分；
- 多样性、新颖性与不确定性；
- Pareto 前沿及最终候选排序；
- 经反馈引导后生成分布的变化。

## 可检验假设

### H1：多源靶点条件

PLM + 3D 口袋图的联合条件相较单独使用口袋几何，能提高 held-out target 上的有效结合候选比例；已知配体约束应作为可选增强，不得成为数据泄漏通道。

### H2：跨模态 PLI 反馈

口袋级蛋白–分子对齐得分相较全蛋白池化或单一 docking 分数，能更可靠地重排生成候选，并在冷靶点上保持更好的校准。

### H3：多目标 Pareto 反馈

Pareto 选择相较单一加权和及逐项阈值过滤，能保留更多性质权衡合理、结构多样的候选。

### H4：生成闭环

使用偏好或奖励反馈引导的生成器，相较 TargetDiff、post-filter 和 Pareto reranking，在相同采样预算下产生更高比例的 Pareto 合格分子。

### H5：未见靶点泛化

按蛋白序列相似性、口袋相似性和分子骨架同时控制泄漏后，方法在 held-out target 和 unseen-family 上仍保持可测的收益，而非只记忆同源靶点或已知配体。

## 固定对照

- TargetDiff；
- TargetDiff + post-filter；
- TargetDiff + Pareto reranking；
- feedback-guided TargetDiff；
- 条件消融：pocket only、PLM only、pocket + PLM、pocket + PLM + known ligand；
- 反馈消融：PLI、ADMET、physics、SA 逐项移除；
- 选择消融：weighted sum、hard filter、Pareto。

## 不作出的结论

- 计算评分不等于实验测定的结合亲和力或安全性；
- docking/PLI 高分不等于真实药效；
- EGFR 突变体间迁移不等于跨蛋白家族泛化；
- 没有真实合成与湿实验时，不宣称获得可开发药物；
- 仅完成生成后筛选时，不宣称实现了反馈引导生成。
