# EGFR target bundle

This directory is generated by:

```bash
python scripts/download_egfr_data.py
```

## Current contents

- UniProt canonical accession: `P00533`
- Canonical sequence length: 1210 aa
- AlphaFold model: `AF-P00533-F1`, model v6 at retrieval time
- Experimental structures:
  - `4G5J`: WT, afatinib complex
  - `2ITZ`: L858R, gefitinib complex
  - `2JIU`: T790M, AEE788 complex
  - `7VRA`: T790M/C797S, HC5476 complex

`7VRA` is intentionally labelled as a double mutant. It is not evidence for a
single-C797S structure.

## Versioned files

- `egfr_structure_manifest.csv`: compact experimental-structure manifest
- `provenance.json`: retrieval time, URLs, versions, and hashes
- this README and empty directory markers

Downloaded FASTA, API JSON, PDB, and mmCIF files are reproducible local data and
are excluded from Git.

## Important limitation

The full-length AlphaFold model is useful for sequence/structure representation,
but its kinase-pocket geometry is not automatically preferred over ligand-bound
experimental structures. Pocket extraction and generation will start from the
co-crystal structures and preserve ligand/covalent-binding annotations.
